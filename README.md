# NoDa Kalkulator

![License](https://img.shields.io/badge/license-All%20Rights%20Reserved-red)

Lønnskalkulator for dansekunstnere i Norge. Beregn rettferdig lønn basert på gjeldende tariffavtaler og veiledende satser fra Norske Dansekunstnere.

**[Live Demo →](https://noda-kalkulator.vercel.app)**

---

## Om prosjektet

NoDa-kalkulatoren er et verktøy som hjelper dansekunstnere å beregne rettferdig lønn. Den er utviklet for å gjøre det enkelt å navigere komplekse tariffavtaler og veiledende satser.

### Hvem er det for?

- Frilans dansekunstnere som forhandler honorar
- Koreografer som planlegger prosjektbudsjetter
- Dansepedagoger som skal sette undervisningspriser
- Produsenter og arrangører som trenger kostnadsoverslag

## Funksjoner

| Kalkulator | Beskrivelse |
|------------|-------------|
| **Danser** | Beregn honorar for prøver og forestillinger |
| **Koreograf** | Kalkuler koreografhonorar basert på prosjektomfang |
| **Danser & Koreograf** | Kombinert kalkulator for doble roller |
| **Pedagog** | Beregn undervisningshonorar |

## Kom i gang

### Forutsetninger

- Node.js 20.9+ (påkrevd for Next.js 16)
- npm eller yarn

### Installasjon

```bash
# Klon repositoriet
git clone https://github.com/[username]/NoDa-kalkulator.git
cd NoDa-kalkulator

# Installer avhengigheter
npm install

# Start utviklingsserver
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000) i nettleseren.
